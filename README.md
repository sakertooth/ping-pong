# Ping Pong
Ping Pong using SFML + TGUI

# Requirements
* C++17
* CMake 3.8>=
* SFML 2
* TGUI
* effolkronium/random

# Credits
* Sounds - [OpenGameArt](https://opengameart.org/content/3-ping-pong-sounds-8-bit-style)
* Font - [Pixeboy](https://www.dafont.com/pixeboy.font)
